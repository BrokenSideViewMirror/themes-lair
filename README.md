# Give Up GitHub

This project has given up GitHub.  ([See Software Freedom Conservancy's *Give Up  GitHub* site for details](https://GiveUpGitHub.org).)

This project will soon be moving to Sourcehut. Don't worry, you can still access this theme through the official kitty-themes repository! However, I will not develop any more projects on Github.

All theme directories, in support of the SFC's *Give Up Github* campaign, will be deleted and rehosted on Sourcehut.

Any use of this project's code by GitHub Copilot, past or present, is done without our permission.  We do not consent to GitHub's use of this project's code in Copilot.

Join us; you can [give up GitHub](https://GiveUpGitHub.org) too!
